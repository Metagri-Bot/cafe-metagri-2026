# cafe-metagri-2026

カフェMetagri 2026 LPの静的サイトです。

## Pages

- `index.html`: LP全体ハブ
- `co-creators.html`: 共創者募集ページ
- `support.html`: 支援者応募・リターン紹介の準備ページ
- `day-guide.html`: 実施当日の案内の準備ページ
- `report.html`: 実施後の事後報告の準備ページ
- `terms.html`: 販売規約の準備ページ
